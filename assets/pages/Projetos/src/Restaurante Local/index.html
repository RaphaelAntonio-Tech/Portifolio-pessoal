Restaurante Local

HTML
CSS

“Sabor & Arte” foi meu primeiro projeto desenvolvido com apenas HTML e CSS, com o objetivo de consolidar o aprendizado sobre a estrutura de um site para um restaurante local fictício. A proposta era criar uma landing page simples, informativa e funcional, utilizando os conhecimentos adquiridos ao longo do módulo do curso de HTML e CSS do Curso em Vídeo promovido pelo professor "Gustavo Guanabara”. Durante o desenvolvimento, também foram trabalhadas diversas soft skills, como perseverança, foco, disciplina, organização e persistência. Esse projeto marcou minha jornada proporcionando a criatividade para futuros projetos práticos.

“Sabor & Arte” foi meu primeiro projeto desenvolvido com HTML e CSS, com o objetivo de consolidar conhecimentos fundamentais em estruturação de páginas web e estilização responsiva. Inspirado em um restaurante local fictício, o site foi construído como uma Landing Page simples, funcional e informativa, durante o módulo de HTML e CSS do Curso em Vídeo, do professor Gustavo Guanabara.

Objetivo:

  • Praticar e consolidar fundamentos de HTML e CSS.
  • Exercitar versionamento com Git e publicação no GitHub Pages.
  • Desenvolver disciplina, organização e foco durante o processo de aprendizado.

Tecnologias Utilizadas:

  • HTML5 – Estrutura semântica das páginas.
  • CSS3 – Estilização visual e responsividade básica.
  • Git & GitHub – Controle de versão e publicação online.

Destaques do Projeto:

  • Seção de apresentação do restaurante (nome, slogan e identidade visual).
  • Cardápio ilustrativo com pratos fictícios.
  • Estrutura clara, layout limpo e responsividade básica para diferentes telas.

Soft Skills Desenvolvidas:

  • Perseverança – conclusão do primeiro projeto prático mesmo diante de desafios.
  • Foco – dedicação ao aprendizado contínuo de HTML e CSS.
  • Organização – estruturação clara da página e do repositório.
  • Disciplina – prática constante ao longo do curso.
  • Criatividade – desenvolvimento de identidade visual fictícia para o restaurante.

Projeto inicial que marcou o início da minha jornada prática em desenvolvimento Front-End, unindo fundamentos técnicos a soft skills como perseverança, foco e organização, além de simular o ciclo de desenvolvimento e entrega de um site real.

Contador Simples

Outubro 2025

HTML

CSS

JavaScript

Figma

github

Avançado meus estudos em JavaScript, desenvolvi o mini projeto Contador Simples com While, que me ajudou praticar conceitos fundamentais de estrutura de repetição e também experimentar novas ferramentos de design como o Figma

Como funciona:

  • O usuário informa um número inicial e um número final.
  • O programa faz a contagem a partir do número inicial até o final.
  • Caso o número inicial seja maior que o final (ex.: iniciar em 20 e terminar em 1), o contador funciona de forma decrescente, até chegar ao destino.

Esse projeto marcou um passo importante para mim, pois foi a primeira vez que utilizei o Figma no processo de criação. Com ele, consegui estruturar uma interface limpa, organizada e intuitiva, pensando não só no funcionamento, mas também na experiência do usuário.

Principais aprendizados técnicos:

  • Aplicação prática do laço de repetição While
  • Melhoria no uso de condicionais para tratar diferentes cenários
  • Estruturação de um fluxo lógico claro para garantir resultados corretos.
  • Integração entre lógica de programação e design da interface.

Soft skills desenvolvidas:

  • Organização do planejamento do projeto
  • Atenção aos detalhes para criar uma interface mais agradável
  • Pensamento lógico aliado à criatividade
  • Perseverança para ajustar erros e melhorar o código

Esse projeto, apesar de simples, foi um marco importante no meu aprendizado. Ele mostra como pequenos exercícios podem gerar grandes evoluções, tanto na parte técnica quanto o desenvolvimento pessoal.

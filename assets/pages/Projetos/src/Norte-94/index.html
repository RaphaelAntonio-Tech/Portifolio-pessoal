Projeto de Landing Page desenvolvido como primeira experiência prática com JavaScript, simulando uma loja fictícia de roupas e calçados femininos. A proposta foi criar uma interface simples, funcional e responsiva, aplicando conceitos iniciais de desenvolvimento Front-End.

Objetivo:

  • Iniciar a prática de JavaScript puro em um projeto real.
  • Desenvolver uma estrutura clara e organizada para uma página de e-commerce.
  • Explorar fundamentos de HTML, CSS e interatividade com JavaScript.

Tecnologias Utilizadas:

  • HTML5 - Estrutura da página e semântica.
  • CSS3 - Estilização, responsividade e layout limpo.
  • JavaScript (Vanilla) – Funcionalidades interativas sem frameworks.

Destaques do Projeto:

  • Navegação clara e intuitiva.
  • Seções bem definidas para banners, produtos e informações.
  • Interações básicas em JavaScript, incluindo botões, rolagem e exibição dinâmica de conteúdo.
  • Aplicação de boas práticas de layout e estilização semântica, voltadas para experiência do usuário.

Soft Skills Desenvolvidas:

  • Disciplina - foco em aplicar fundamentos técnicos com consistência.
  • Organização - estruturação clara do layout e código.
  • Aprendizado contínuo - primeiras práticas com lógica de programação.
  • Persistência - superação de erros e testes de interatividade.
  • Atenção aos detalhes – cuidado na navegação e usabilidade do site.

Este projeto marca o início da minha jornada prática como desenvolvedor Front-End, unindo teoria e prática em uma aplicação simples, mas realista, no contexto de e-commerce.

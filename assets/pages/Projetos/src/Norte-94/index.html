"Norte 94" foi meu segundo projeto e meu primeiro projeto utilizando JavaScript, com o objetivo de criar uma Landing Page simples para uma loja de roupas fictícia. A proposta era criar uma página simples, utilizando HTML, CSS e JavaScript e conceitos básico de responsividade. Esse projeto marcou uma etapa importante na jornada de aprendizado, proporcionando uma experiência prática valiosa e reforçando a base para projetos mais avançados no futuro.

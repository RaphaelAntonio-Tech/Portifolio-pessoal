Curiosidades de tecnologia

Abril 2025

HTML

github

Desenvolvido durante a conclusão do módulo 2 do curso de HTML5 e CSS3 do Curso em Vídeo, este projeto teve como objetivo principal consolidar conhecimentos fundamentais sobre estruturação web. O foco foi compreender profundamente como funciona a arquitetura de um site moderno e aplicar esses conceitos de forma prática e eficiente..

A proposta central do projeto consistiu em criar uma página web completa, combinando aspectos informativos com uma interface funcional e atraente. Foram implementadas diversas técnicas de HTML5 incluindo:

  • Estruturação hierárquica de títulos (h1-h6)
  • Organização semântica com tags como header ,section ,article e footer.
  • Implementação de listas ordenadas e não-ordenadas
  • Criação de links internos e externos com âncoras apropriadas
  • Inserção e otimização de imagens responsivas
  • Utilização estratégica de classes e IDs para seleção CSS
  • Aplicação de estilos personalizados via CSS externo

O conteúdo temático do site, focado em curiosidades tecnológicas, permitiu uma abordagem criativa na apresentação visual, com gradientes personalizados, tipografia cuidadosamente selecionada e um esquema de cores harmonioso que valoriza a experiência do usuário. Durante o processo de desenvolvimento, diversas soft skills foram aprimoradas, incluindo:

  • Comunicação escrita clara e concisa
  • Pensamento lógico na estruturação do código
  • Foco e disciplina para concluir cada etapa
  • Autonomia na resolução de problemas técnicos
  • Criatividade na apresentação visual e organização do conteúdo
  • Organização metódica dos arquivos e componentes do projeto

Este projeto representou um marco significativo em minha jornada de aprendizado, proporcionando uma experiência prática valiosa com desafios reais de desenvolvimento web. Os conhecimentos adquiridos e consolidados estabeleceram uma base sólida para a construção de projetos mais complexos e sofisticados no futuro.

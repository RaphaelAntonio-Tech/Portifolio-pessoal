Entre Linhas & Cores

Maio 2025 - Junho 2025

HTML

CSS

JavaScript

github

“Entre Linhas & Cores" é uma galeria de arte fictícia desenvolvida como projeto pessoal, com o objetivo de unir estética, navegação intuitiva e valorização da arte visual em um ambiente digital. A galeria foi desenvolvida com HTML, CSS e JavaScript, buscando boas práticas de acessibilidade,sabilidade e performance.

Objetivo:

  • Exercitar habilidades em desenvolvimento Front-End
  • Aplicar boas práticas de semântica, acessibilidade e performance.
  • Simular a experiência de navegação em uma galeria de arte online.

Tecnologias Utilizadas:

  • HTML5 - Estrutura semântica das páginas.
  • CSS3 - Estilização responsiva e organização visual.
  • JavaScript – Interatividade e melhorias na navegação.

Destaques do Projeto:

  • Interface com navegação intuitiva e design responsivo, adaptado para diferentes dispositivos.
  • Atenção à performance, otimizando carregamento e usabilidade.

Soft Skills Desenvolvidas:

  • Criatividade - elaboração de soluções visuais e estéticas.
  • Organização - estruturação clara do projeto e do código.
  • Persistência - superação de dificuldades durante o desenvolvimento.
  • Atenção aos detalhes - foco em usabilidade, acessibilidade e design.
  • Autonomia – condução do projeto de forma independente, do conceito à entrega.
